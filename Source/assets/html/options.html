The Image Filter scans image names and search tags for any of the 'Blocked Filenames'. If a match is found, the image will be replaced. Partial word matches are detected. If a listed word appears as part of a word that is in the image name or search tag, then the image will still be blocked. In other words, if you listed 'porn' as a blocked word, then the word 'pornography' will also be checked, since the word 'porn' appears in 'pornography'.
